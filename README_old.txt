
SAE Math.AI

Pengenalan:
SAE Math.AI ialah sebuah aplikasi AI yang dibangunkan untuk membantu menyelesaikan soalan Matematik sekolah rendah. Nama "SAE" diambil daripada huruf awal nama ahli kumpulan, iaitu Shaiful, Adib dan Elman. Projek ini menggunakan Python dan Streamlit untuk membina antara muka web yang interaktif dan mesra pengguna.

Ciri-ciri Utama:
- Menyelesaikan soalan dalam bentuk teks biasa seperti “Berapakah 1/2 tambah 3/4?”
- Menyokong pelbagai operasi matematik: pecahan, perpuluhan, peratus, nisbah, bundar, tambah, tolak, darab, bahagi
- Memberikan langkah penyelesaian secara berperingkat
- Antaramuka mesra pengguna menggunakan Streamlit
- Logo dan nama AI terpapar di atas
- Log perbualan (chat log) automatik lengkap dengan tajuk yang dijana sendiri berdasarkan perbualan
- Input mesej diletakkan di bawah seperti sistem mesej sebenar
- Sokongan input suara dan gambar (jika disambungkan dengan sambungan lanjut)
- Boleh simpan nota/jawapan sebagai fail PDF

Cara Menjalankan Aplikasi:
1. Pastikan anda telah memasang Python (dicadangkan Python 3.10 dan ke atas)
2. Pasang keperluan menggunakan pip:
   pip install streamlit sympy
3. Jalankan aplikasi dengan arahan berikut di command prompt:
   streamlit run app.py
4. Akses aplikasi di pelayar web (biasanya http://localhost:8501)
